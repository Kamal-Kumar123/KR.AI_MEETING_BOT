Privacy Policy

KRAI Meeting Assistant (browser extension & web app)
Last updated: July 9, 2026

This Privacy Policy describes how KRAI Meeting Assistant ("we", "our", "the Service") collects, uses, stores, and protects information when you use our Chrome/Edge extension and website.

1. Information we collect

  • Account information: email address, name (when you register or sign in with Google), and an authentication token stored locally in your browser.
  • Meeting audio: audio you explicitly start recording from supported meeting platforms (Google Meet, Microsoft Teams, Zoom) or upload manually.
  • Meeting content: transcripts, speaker labels, summaries, action items, and related metadata (platform, duration, timestamps, meeting title).
  • Configuration: API URL and frontend URL you set in extension options (stored in browser sync/local storage).
  • Website content: when recording is active, audio from the meeting tab and/or microphone you approve may be captured and uploaded for transcription.

2. How we use information

  • Authenticate you and provide access to your meetings dashboard.
  • Transcribe recordings using cloud speech-to-text (Deepgram).
  • Generate summaries, titles, and action items using cloud AI (Google Gemini).
  • Store recordings and results so you can view, share, and delete them.
  • Improve connected-meeting features using optional RAG context within your project series (only when you choose "connected" mode).

3. Recording consent

The extension does not record silently. Recording starts only when you click Start Recording. You are responsible for informing meeting participants and obtaining consent required by your organization or local laws before recording.

4. Where data is stored

  • Cloud database: account and meeting metadata (PostgreSQL via Neon).
  • Cloud object storage: audio files (Cloudflare R2 or compatible S3 storage).
  • Your browser: login token and extension settings (Chrome storage API).
  • Vector search (optional): meeting embeddings for connected-series RAG (ChromaDB on the API server).

5. Third-party services

We use trusted processors to operate the Service. Data is sent only to provide features you request:

  • Google OAuth — sign-in (email, profile).
  • Deepgram — speech-to-text transcription.
  • Google Gemini API — summaries and action items.
  • Cloud hosting — Render (API), Vercel (website), Cloudflare (storage).

We do not sell or rent your personal data to third parties.

6. Data sharing

  • We do not sell user data.
  • We do not use user data for unrelated purposes (such as advertising or credit scoring).
  • Share links you create may expose meeting summaries/transcripts to anyone with the link and share token.
  • We may disclose data if required by law.

7. Data retention & deletion

Meeting data is kept until you delete it from My Meetings in the web app or via the API. Deletion removes associated database records and stored audio files. You may delete your account by contacting us.

8. Security

We use HTTPS, authenticated API access (JWT), and cloud provider security controls. No method of transmission over the Internet is 100% secure; use strong passwords and protect your device.

9. Children

The Service is not intended for users under 13 years of age.

10. Changes

We may update this policy. The "Last updated" date at the top will change when we do.
